---
title: "include"
---

Directs the executor to include this field or fragment only when the `if` argument is true.

```
directive @include(
  if: Boolean!
)

```

### Arguments[](#arguments "Direct link to heading")

#### `if` ([Boolean](/developer/graphql-api/references/scalars/boolean))[](#if-boolean "Direct link to heading")

Included when true.
